Sistema de Votação Eletrónica C#
Sistema completo de votação digital em C# (.NET Console) com autenticação, 3 tipos de eleições, auditoria, notificações e relatórios.

Funcionalidades
3 Eleições: Presidencial (50%), Assembleia (33%), Municipal (25%)
Login Seguro: 3 tentativas, bloqueio 5min
Relatórios + Gráficos ASCII
Notificações (4 níveis)
Auditoria completa
Auto-save/Backup
